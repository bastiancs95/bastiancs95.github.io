Proyectos

Tres casos con el mismo formato: el problema, lo que construí y el resultado medido.

Reporting ejecutivo B2B en tiempo real

BG Chile · 2025 — presente

Contexto. Cada reunión de gerencia exigía 2–3 días armando informes desde planillas dispersas, y los márgenes de importación estaban repartidos entre múltiples fuentes sin una vista común.

Qué construí. Un modelo de datos y dashboards ejecutivos en Power BI (DAX, Power Query) con actualización automática, más cost breakdowns de importaciones aéreas y marítimas que dejan el margen de cada producto visible al día.

Resultado. Los informes para gerencia pasaron de 2–3 días a menos de 2 horas, y las decisiones de compra se toman con márgenes actualizados en vez de cierres del mes anterior.

Power BI · DAX · Power Query · SQL

Automatización de reportería multi-área

VNGS Consultorías · 2024 — 2025

Contexto. Marketing, comercial y operaciones armaban sus reportes a mano cada semana, cada área con su propia planilla y sus propios criterios de cálculo.

Qué construí. Pipelines ETL en SQL y Python que consolidan las fuentes, y un dashboard en Power BI por área con KPIs comunes de campañas: CAC, tasa de conversión y ROI.

Resultado. Unas 8 horas semanales menos de trabajo manual por área (~30%) y campañas que se corrigen con datos frescos: +12,5% de ROI en Q2-2024.

Power BI · SQL · Python · ETL

Automatización documental con agentes de IA

Proyecto personal · 2026

Contexto. Un proceso personal recurrente exigía revisar decenas de páginas web, contrastar su contenido con criterios propios y producir documentos personalizados para cada caso — varias horas de trabajo manual al día.

Qué construí. Un sistema de agentes con Claude Code que navega y extrae datos con Playwright, los estructura y puntúa contra una rúbrica configurable, y genera documentos PDF personalizados vía LaTeX; N8N orquesta las notificaciones y el seguimiento queda registrado en una planilla de tracking. Cada salida pasa por revisión humana antes de usarse.

Resultado. El ciclo completo pasó de horas a minutos por documento, con calidad consistente. Lo uso a diario desde junio de 2026.

Claude Code · Playwright · N8N · Python · LaTeX
